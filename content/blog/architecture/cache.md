# 캐쉬, Cache

Cache는 원래 (특히 해적이) 보물이나 식량을 감춰두는 공간을 뜻하는 말에서 유래됐다. 컴퓨터에서 캐시는 필요한 데이터를 미리 저장 (buffering) 해두는 SW,HW 요소. Memory Hierarchy에서 중요한 역할을 함. Latency와 Throughput 모두를 증가시킬 수 있다.
